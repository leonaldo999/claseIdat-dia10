Sintaxis

      
        h1 {
          color: red;
        }

  • h1: Selectores

  • color: Propiedad

  • red: Valor

Selectores

        
          /* SELECTOR DE ETIQUETA */
            h1 {
              color: red;
            }
          /* SELECTOR DE DESCENDENCIA */
            nav ul li a {
              color: red;
            }

Cascada

Significa que los estilos que llegan en ultimo lugar , sobreescriben a los de antes.
La especificidad vence a la cascada 

      
        h1 {
          color: red;
        }
        
        h1 {
          color: blue;
        }

Especificidad

Selector de Etiqueta (1)
Selector de Clase (10)
Selector de ID (100)
INLINE (1000)
IMPORTANT (infinito) 

        
          /* Selector de Etiqueta (4) */
            nav ul li a {
              color: red;
            }

          /* Selector de Etiqueta (1) */
            a {
              color: blue;
            }

          /* INLINE (1000) */
          
CSS Cascade Style Sheet


          /* ESPECIFICIDAD: IMPORTANT (infinito) */
          
            .padre #hijo {
              color: red;
            }

            h1 {
              color: blue !important;
            }

          /* ESPECIFICIDAD: clase y ID = 110 */
            .padre #hijo {
              color: red;
            }
